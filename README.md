# 📊 Power BI Learning & Dashboard Projects

## 📌 Overview

This repository showcases my Power BI learning journey, including dashboard creation, data transformation, and data visualization techniques using real-world datasets.

### 🔹 Data Preparation

* Data cleaning
* Handling missing values
* Removing duplicates
* Data type conversion

### 🔹 Data Transformation (Power Query)

* Filtering & sorting
* Merging & appending queries
* Column transformations

### 🔹 Data Modeling

* Relationships between tables
* Star schema basics

### 🔹 DAX (Data Analysis Expressions)

* Calculated columns
* Measures (SUM, AVERAGE, COUNT)
* Advanced functions (SUMX, DIVIDE)

### 🔹 Data Visualization

* Bar charts
* Line charts
* Pie charts
* Cards & KPIs

### 🔹 Dashboard Development

* Sales Dashboard
* Customer Dashboard
* Inventory Dashboard
* Forecast Dashboard
